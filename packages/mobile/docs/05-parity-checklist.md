# Mobile parity checklist

## Implemented
- Pan, pinch zoom, and double-tap fit to view.
- Node drag + tap selection.
- Port-to-port edge creation with live preview + snap.
- Handoff animations on edges.
- Minimap navigation.
- Node inspector (chat + tool timeline).
- Approval queue actions.

## Missing
- Edge selection + deletion.
- Multi-select or box selection.
- Node context actions (duplicate, delete).
- Run controls (start/pause, global mode toggle).
- Visual parity for edge labels and advanced styling.

## Stability
- iOS and Android smoke testing required each release.
